### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（会员）、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 电影信息
5. 影院商品
6. 通知公告
7. 后台管理
8. 在线咨询
9. 个人中心
10. 我的订单
11. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 会员管理
4. 电影分类管理
5. 电影信息管理
6. 影院商品管理
7. 商品分类管理
8. 影厅信息管理
9. 会员卡管理
10. 挂失信息管理
11. 系统管理（系统简介、通知公告、在线咨询、关于我们、轮播图管理）
12. 订单管理
